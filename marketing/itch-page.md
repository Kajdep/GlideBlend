# GlideBlend itch.io Page Copy

## Short Description
Blend AI video clips with perceptual frame matching and export a cleaner transition in one desktop tool.

## Tagline
A desktop utility for stitching generated shots without the ugly hard cut.

## Long Description
GlideBlend helps creators join two clips at the closest visual match instead of cutting at a random timestamp. It scans the end of clip one and the start of clip two, compares the frames with perceptual hashing, then trims and joins the videos locally on your machine.

This is built for AI-video workflows like Sora, Veo, Runway, Pika, and shot-based animation pipelines, but it also works on ordinary video files when you want a faster first-pass transition.

Why it sells well on itch.io:
- Fast and focused. It solves one painful post step instead of pretending to be a full editor.
- Local-first. Your footage stays on the machine.
- Creator-friendly. Chain results, test variations, and save a merged MP4 without bouncing between tools.

Core features:
- Best-match frame search using 64-bit dHash and Hamming distance
- Local FFmpeg processing with no upload requirement
- Chain the last output into the next merge
- Desktop save dialog for final MP4 export
- Step-by-step guide built into the app
